# Lattes JCR Parser

Aplicação web client-side para extração do fator de impacto (JCR) a partir de arquivos HTML do Currículo Lattes.

## Uso
1. Salve o currículo como **Página completa (HTML)**.
2. Faça upload no app.
3. Exporte os dados para Excel.

Nenhum dado é enviado ou armazenado externamente.
