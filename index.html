Lattes JCR Parser

Envie o arquivo HTML completo do Currículo Lattes (salvo como “Página completa”, preferencialmente via Firefox). O processamento ocorre localmente no seu navegador.


Dica: arquivos .mhtml não funcionam. Salve como HTML completo.

Pré-visualização

Observação: o JCR é extraído do tooltip do ícone “jcr” no HTML do Lattes. Se o HTML não contiver esses ícones, as colunas de JCR ficarão vazias.
